The project was the first one in the course, which dealth with design and analysis of algorithms and focused on the use of Data Structures as well. It was divided into 3 parts:

Part 1: Desiging Validation Algorithm


See the above specificaiton for more details.

This part required us to develop an algorithm to validate or parse a mathematical expression, and if it were valid, return the final result. This was accomplished using 2 stacks, one for operators and one for values. As soon as we came accross an ill-formed expression, a NotWellFormed error was raised. As and when we encountered brackets, the expression enclosed in the bracket was performed and the values on the stack replaced by the result of the expression. This was done recursively.

Part 2: Deque Data Structure


See the above specificaiton for more details.

This part required us to develop and implement algorithms that performed 3 tasks when it came to the deque structure. The tasks were:
  • Reversal using Iteration
  • Reversal using Recursion
  • Splitting while maintaining stability

Part 3: Can I ski?


See the above specificaiton for more details.

This part required us to process input for a Directed Acyclic Graph (DAG) from stdin and then determine if each node could be visited in one run.
